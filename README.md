Petro Operaciones Release Notes
========================

Updates 1.4.1
---------------------
* Se excluye los seguimientos de los clientes en el reporte de seguimientos.
* Se autocompleta el formuluario de cotización al seleccionar un cliente existente.

Updates 1.4.0
---------------------
* Se incluye el módulo para consulta operaciones, seguimientos y documentos por parte de los clientes


Updates 1.3.0
---------------------
* Se incluye el check list para realizar el proceso de finalización de un Documento de Operaciones

Updates 1.2.2
---------------------
* Se soluciona el bug en total de operaciones por cliente

Updates 1.2.1
---------------------
* Se incluye filtro año para el reporte Total de Operaciones
* Se incluye porcentaje de seguimientos por usuario en el reporte Total de Operaciones
* Se quitan del reporte Total de Operaciones las personas que no tengan seguimientos
* Se puede editar el nombre del cliente en las prefacturas

Updates 1.2.0
---------------------
* Se amplia la logitud de los campos No de Documento a 30 carácteres.
* Se modifica el titulo en https://www.pcilogistic.co/petrocargo/Reports/TotalMercanciaMovilizada,  Total Operaciones por Modo de Transporte
* PCI permite crear, editar y eliminar descripción de operaciones, ver el menú Parametrización
* Se modifica el formato de factura para que la impresión quede día mes año
* Se modifica el formato de pre factura para que la impresión quede día mes año

Updates 1.1.0
---------------------
* Se amplia la logitud de los campos BL HBL AWB HAWB a 30 carácteres.
* Se cambia el nombre de operación por descripción de la operación
* Se crea un nuevo campo llamado Modalidad en Documentos de Operaciones
* Se crea el reporte de Operaciones por Modalidad
* Se modifica la modalidad operativa por tipo de operación
* Se modifican los tipos de operación a 
		Agenciamiento de Carga
		Logistica Nacional
		Logistica Integral
		Agenciamiento Aduanero
